## How access works

1. Review the dataset description, release notes, population, and file list on its catalog page.
2. Select **Access files in Globus**.
3. Sign in when Globus asks you to authenticate.
4. Download an HTTPS-enabled file or transfer larger data to a Globus collection you can access.

Adding another dataset does not require a new application or backend. Copy the starter detail page, edit the metadata, add it to this index, and point its Globus link to the correct collection folder.
