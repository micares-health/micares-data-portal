## Publishing document files

Public PDFs, spreadsheets, and other documentation files can be placed under:

`content/assets/documentation/`

The portal build copies files from `content/assets/` into the published site. For example:

`content/assets/documentation/baseline-codebook.pdf`

can be linked as:

`/documentation/baseline-codebook.pdf`

Do not place restricted or participant-level information in the public repository. Controlled documentation can instead remain in Globus and use a folder-specific Globus link.
